# 论文数学公式修正总结

## 已修正的问题

### 1. 全局损失函数权重 (section_system.tex)
**修正前**: `F(w) = Σ (p_n/N) F_n(w)`
**修正后**: `F(w) = Σ p_n F_n(w)`
**原因**: p_n已经是归一化权重，不应再除以N

### 2. 能量消耗模型 (section_system.tex)
**修正前**: `E_n(t) = σ²/|h_n(t)|²`
**修正后**: `E_n(t) = β·σ²/|h_n(t)|²`
**原因**: 添加缩放因子β，包含传输功率、模型大小等参数

### 3. 差分隐私预算 (section_system.tex)
**修正前**: `ε_total ≈ O(√(T ln(1/δ))/(σ_DP·K))`
**修正后**: `ε_total ≈ q√(2T ln(1.25/δ))/σ_DP`，其中q=K/N
**原因**: 使用标准moments accountant公式

### 4. Lyapunov虚拟队列 (section_problem.tex)
**修正前**: `Q_n(t+1) = max{Q_n(t) + E_target - E_remain,n(t), 0}`
**修正后**: `Q_n(t+1) = max{Q_n(t) + (E_target - E_remain,n(t+1)), 0}`
**原因**: 使用更新后的能量值，逻辑更清晰

### 5. Lyapunov推导说明 (section_problem.tex)
**改进**: 添加了drift定义和权重公式的直观解释

### 6. DP噪声方差说明 (section_method.tex)
**改进**: 明确噪声与敏感度C/K的关系，修正隐私预算计算

### 7. 算法伪代码 (section_method.tex)
**改进**:
- 能量公式添加β因子
- 虚拟队列更新独立循环，覆盖所有客户端
